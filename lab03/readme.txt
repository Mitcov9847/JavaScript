1. Доступ к элементу на веб-странице с помощью JavaScript можно получить с помощью метода document.getElementById, document.querySelector
 или с помощью обхода DOM-дерева, используя методы как document.getElementsByTagName, ocument.getElementsByClassName и другие.

2. Делегирование событий - это механизм обработки событий в DOM, при котором события обрабатываются не на самом элементе, 
на котором они произошли, а на его родителе или более высоком уровне в DOM. Это позволяет обрабатывать события на динамически 
созданных элементах, а также повышает производительность, так как обработчик событий устанавливается на общего родителя, 
а не на каждый отдельный элемент.

3. Для изменения содержимого элемента DOM с помощью JavaScript можно использовать свойства элемента, такие как `innerHTML, textContent,
 setAttribute и другие.

4. Новый элемент в DOM-дерево можно добавить с помощью метода createElement для создания нового элемента, а затем метода appendChild
 или insertBefore
для добавления этого элемента в дерево, указывая его родителя.
